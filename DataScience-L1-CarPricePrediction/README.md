# Car Price Prediction using Machine Learning

## Objective
Predict the selling price of a used car using machine learning.

## Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Models

- Random Forest Regressor
- Gradient Boosting Regressor

- MAE

- R� Score
